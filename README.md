# dotFiles

## vim

### セットアップ

+ ディレクトリの作成
```
$ mkdir ~/.vim
$ mkdir ~/.vim/bundle
$ mkdir ~/.vim/tmp
$ mkdir ~/.vim/tmp/undo
$ mkdir ~/.vim/tmp/bkup
```

+ neobundleのインストール
```
$ git clone https://github.com/Shougo/neobundle.vim ~/.vim/bundle/neobundle.vim
```

上記設定後、vimを起動することで、~/.vimrcの設定に従い、各種プラグインがインストールされる。
